Hotel Booking System

Overview

A Hotel Booking System developed as part of the Data Structures (CS212T) course. The project applies fundamental data structures to efficiently manage hotel operations, including guest records, room bookings, and payment processing.

Features

Manage guest information.
Book and manage hotel rooms.
Search and organize booking records.
Process and track payments.

Data Structures Used

Doubly Linked List – Guest management.
Binary Search Tree (BST) – Room and booking management.
Stack – Payment processing.

Technologies

Java
Object-Oriented Programming (OOP)
Data Structures
